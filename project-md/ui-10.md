## /[org]/settings - 组织设置

### 左侧子导航（Sidebar）
- General
- Domains
- Tokens
- Webhooks
- Environment Variables (Org-level)
- SSO（Enterprise）
- Security
- Danger Zone

### General
- Organization Name（Input + Save）
- Slug（带 URL 预览：deployx.app/{slug}）
- Logo 上传（Avatar + 拖拽上传区）
- 默认 Region 选择

### Tokens
- 已生成 Token 列表（Table）
  - Name / Scopes Badges / Created / Last Used / Expires / Revoke
- "Create Token" 按钮 → Dialog
  - Name / Scopes 多选 / 过期时间（Never/30d/90d/1y/Custom）
  - 创建后**仅显示一次**完整 token（Alert 警告 + 复制按钮）

### Webhooks
- Webhook 列表（URL + 事件 Badge + 状态）
- 创建 Webhook：URL / 事件多选（deployment.created/succeeded/failed/...）/ Secret / Active

### Security
- 强制 2FA Switch
- Session 列表（设备 / IP / 最后活动 / Revoke）
- IP 白名单（Pro+）
- SAML SSO 配置（Enterprise）

### Danger Zone
- 红色边框卡片
- Transfer Ownership / Delete Organization
- 每个操作需 AlertDialog 二次确认（输入组织名匹配）

请使用 Sidebar (shadcn 新增), Form, Table, Dialog, AlertDialog, Switch 等。
输出完整代码。

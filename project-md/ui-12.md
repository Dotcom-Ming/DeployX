请实现高性能 Terminal Log 组件 components/ui/terminal-logs：

### 功能
- 虚拟滚动（react-virtuoso），支持 10w+ 行
- ANSI escape codes 解析（ansi-to-html 或 anser）
- 折叠分组（基于日志中的 #region 标记）
- 实时追加（WebSocket 流）
- 自动滚动至底部 Toggle
- 关键词搜索 + 高亮 + 上下匹配跳转
- 错误行高亮 + 跳到下一个错误
- 复制单行 / 复制全部 / 下载 .log

### Props
{
  logs: LogLine[]
  streaming?: boolean
  onSearch?: (query: string) => void
  height?: number
}

### 样式
- 暗色模式：bg-zinc-950 text-zinc-100
- 行号灰色 mono 字体，hover 高亮整行
- 时间戳灰色 + 级别色彩 + 内容
- 选中行可复制时间戳链接（深链）

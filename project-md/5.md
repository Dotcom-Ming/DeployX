请实现一套完整的 SaaS 计费系统，集成 Stripe：

### 套餐设计
- Hobby（免费）：100GB 带宽、6000 构建分钟、100GB-Hours 函数运行
- Pro（$20/用户/月）：1TB 带宽、24000 构建分钟、1000GB-Hours
- Enterprise（联系销售）：自定义额度 + SSO + SLA

### 计量指标（Metering）
1. Bandwidth：网关层 nginx access log → Vector → ClickHouse 聚合
2. Build Minutes：构建任务开始/结束时间差
3. Function Invocations & Duration：Knative metrics 采集
4. Storage：定时任务统计
5. 数据按 1 小时粒度写入 UsageRecord

### 计费流程
1. 用户订阅 → 创建 Stripe Subscription
2. 每日定时任务聚合用量 → 上报 Stripe Usage Records（按量计费部分）
3. 月底 Stripe 自动出账 → Webhook 同步 Invoice 状态
4. 超额预警：80%、100%、超额时邮件 + 站内通知
5. 欠费冻结：自动停止部署、保留只读访问

### 防作弊
- 用量上限硬限制（请求级拦截）
- 异常流量识别（同 IP 高频、爬虫）
- 免费版限制：10 个项目、3 个成员

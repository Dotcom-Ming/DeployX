请实现权限保护组件 components/auth/can.tsx：

### 用法
<Can permission="project:deploy" fallback={<UpgradePrompt />}>
  <Button>Deploy</Button>
</Can>

<Can role="ADMIN">
  <DangerZone />
</Can>

<Can permissions={["billing:view", "billing:edit"]} mode="all">
  ...
</Can>

### 实现
- 基于 useAuth() hook 获取当前用户权限
- 与后端 Casbin 同步权限映射（前端缓存）
- 支持 role / permission(s) / 自定义 check 函数
- fallback 默认 null，可定制
- loading 状态返回 Skeleton

### 配套
- usePermission(permission) hook 返回 boolean
- 高阶组件 withPermission(Component, permission)

请实现一套企业级认证与权限系统：

### 认证（Authentication）
- 邮箱密码注册/登录（bcrypt + 邮箱验证）
- OAuth2 第三方登录：GitHub / GitLab / Google
- JWT Access Token (15min) + Refresh Token (7d) 双令牌机制
- API Token 用于 CLI 和 CI/CD（带 scope 限制）
- 多因素认证 MFA (TOTP)

### 授权（Authorization）
基于 Casbin 实现 RBAC + ABAC 混合模型：

预设角色：
- OWNER：组织所有权限（含计费、删除组织）
- ADMIN：除计费外所有权限
- DEVELOPER：项目创建、部署、查看日志
- VIEWER：只读
- BILLING_MANAGER：仅计费相关

要求：
1. 提供 NestJS 装饰器 @RequirePermission('project:deploy')
2. 提供资源级权限拦截器（如：用户只能操作所属组织的资源）
3. 提供前端 <Can permission="..."> 组件
4. 所有敏感操作记录到 AuditLog

请输出：
- Casbin 策略 model.conf 和初始 policy.csv
- AuthGuard、PermissionGuard 实现
- 完整的 auth.module.ts、auth.service.ts、auth.controller.ts 代码
